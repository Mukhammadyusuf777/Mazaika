# 🚀 Mazaika — Презентация платформы для инвесторов (Investor Pitch & Tech Brief)

---

## 1. 🎯 Исполнительное резюме (Executive Summary)

**Mazaika** — это инновационная SaaS No-Code платформа следующего поколения, разработанная для быстрой автоматизации продаж, клиентского сервиса и e-commerce в экосистеме **Telegram**.

Платформа позволяет владельцам бизнеса за несколько минут создать профессионального Telegram-бота, интегрированный интернет-магазин (**Telegram Mini App / Web App**) и CRM-систему без написания кода и привлечения программистов.

### 📌 Ключевые показатели продукта:
* **Целевой рынок**: Малый и средний бизнес (SMB), E-commerce, рестораны, салоны красоты, образовательные центры, инфобизнес и сервисы доставки в Узбекистане и СНГ.
* **Решаемая проблема**: Высокие чеки на разработку (от $1 500 до $10 000+), долгие сроки разработки (от 1 до 3 месяцев) и сложность поддержки традиционных мобильных приложений.
* **Наше решение**: Создание полноценной цифровой экосистемы (Бот + Mini App + CRM + Прием платежей) за 15 минут с ежемесячной подпиской от $15/мес.

## 2. ⚡ Уникальные конкурентные преимущества (Competitive Advantages)

1. **Единый движок единого источника правды (Single Source of Truth)**:
   В отличие от существующих конструкторов, где бот и веб-приложение настраиваются отдельно, в Mazaika внедрена **двунаправленная синхронизация**. Изменение каталога товаров или текста в визуальном конструкторе сайта моментально обновляет и WebApp, и кнопку «Menu» в Telegram-боте.

2. **Нативная адаптация под рынок Узбекистана**:
   * Встроена прямая поддержка локальных платежных систем **Payme** и **Click** с автоматической выпиской инвойсов внутри Telegram.
   * Интеграции с профильными CRM-системами **Yclients** (для салонов и клиник) и **GetCourse** (для школ и инфобизнеса).

3. **Google Sites Style No-Code Builder**:
   Интуитивно понятный визуальный блочный конструктор с **живым симулятором смартфона** прямо в браузере. Изменения видны мгновенно в реальном времени.

4. **Enterprise-уровень исполнения сценариев**:
   * **Асинхронный планировщик таймеров**: Устойчив к перезагрузкам серверов. Задержки сообщений и цепочки прогрева не теряются при сбоях.
   * **Математический и JS-движки**: Возможность выполнять вычисления (cashback, скидки, формула стоимости доставки) прямо внутри блоков.

5. **100% Мобильная адаптивность (Mobile-First)**:
   Платформа полностью оптимизирована для управления со смартфона. Предприниматель может редактировать бота, менять цены на товары и отвечать клиентам в CRM-чате прямо с телефона.

## 3. 🏗 Технологический стек и Архитектура (Tech Stack)

Платформа построена по самым современным стандартам веб-разработки с акцентом на высокую производительность, масштабируемость и безопасность:

```
                  +-----------------------------------+
                  |      Пользовательский Фронтенд    |
                  |     (React 18, Vite, TypeScript)  |
                  +-----------------+-----------------+
                                    |
          +-------------------------+-------------------------+
          |                                                   |
+---------v----------+                                 +------v-------------+
| Визуальный Граф    |                                 | No-Code Site       |
| (React Flow)       |                                 | & MiniApp Builder  |
+---------+----------+                                 +------+-------------+
          |                                                   |
          +-------------------------+-------------------------+
                                    |
                  +-----------------v-----------------+
                  |      Серверная система (Backend)   |
                  |    NestJS + Telegraf + Prisma ORM  |
                  +-----------------+-----------------+
                                    |
          +-------------------------+-------------------------+
          |                                                   |
+---------v----------+                                 +------v-------------+
| База Данных        |                                 | Firebase / Firestore|
| PostgreSQL + Redis |                                 | Real-time Sync     |
+--------------------+                                 +--------------------+
```

### 🔹 Frontend (Клиентская часть):
* **Фреймворк**: React 18, Vite, TypeScript.
* **Графовый визуальный движок**: `@xyflow/react` (React Flow) — для отрисовки сложных схем диалогов.
* **Управление состоянием**: Zustand — обеспечивающий молниеносный отклик интерфейса.
* **Дизайн-система**: Кастомная премиальная тема **"Aurora Dark"** (глубокие темные тона, эффекты Glassmorphic, неоновые акценты).

### 🔹 Backend (Серверная часть):
* **Архитектура**: NestJS (TypeScript) с модульной структурой.
* **Telegram Engine**: `Telegraf` SDK с механизмом динамического управления ботами (запуск, остановка и обновление сценариев без перезагрузки сервера).
* **База данных & ORM**: PostgreSQL + Prisma ORM для надежного хранения пользователей, транзакций и истории CRM.
* **Real-time дата-слой**: Firebase Firestore для мгновенной синхронизации конструктора Mini App и веб-сайтов.

## 4. 🧩 Функциональные возможности платформы

### 4.1. Визуальный конструктор сценариев (22 типа блоков)
* **Входные точки (Start)**: Поддержка Deep-linking (реферальные ссылки, рекламные метки).
* **Коммуникация**: Сообщения с медиа (фото/видео), текстовые вопросы, сбор контактов (телефон, email, GPS-локация с автоматической валидацией).
* **Умная логика**: Блок «Shart (If)» для разветвления сценариев, «A/B Тестирование» для конверсий, проверщик подписки на Telegram-каналы.
* **Интеграции**: Вызов Webhooks, Google Sheets (запись и чтение данных), вызов внешних HTTP API.
* **Платежи**: Автоматические инвойсы Payme и Click с блокировкой продвижения по сценарию до факта оплаты.

### 4.2. Конструктор сайтов и Telegram Mini Apps
* **Готовые стили оформления**: *Glassmorphism* (стеклянный стиль), *Minimalist Light* (светлый бизнес-стиль), *Neon Cyberpunk* (яркий стиль).
* **Модульная структура блоков**: Баннеры (Hero), О компании (About), Каталог товаров с корзиной (Catalog), Формы заявок (Form), Виджет кэшбека и баланса (Loyalty), Опросники (Voting).
* **Прямая интеграция в Telegram**: Автоматическая привязка кнопки «Menu» в левом нижнем углу чата бота.

### 4.3. CRM-система и Аналитика
* **Чаты в реальном времени**: Возможность оператора перехватить диалог у бота и ответить клиенту вручную.
* **База подписчиков**: Сегментация контактов, просмотр истории заказов, просмотр баланса и экспорт данных.
* **Аналитика**: Графики активности пользователей и воронка конверсий.

## 5. 💰 Бизнес-модель и Монетизация (Business Model)

Платформа Mazaika использует проверенную гибридную модель монетизации **SaaS (Software-as-a-Service)**:

1. **Тарифные подписки (Recurring Revenue)**:
   * **Free (Бесплатный)**: 1 бот, до 100 подписчиков (для ознакомления).
   * **Pro ($15 / мес)**: До 5 ботов, 5 000 подписчиков, конструктор Mini App, интеграция с Payme/Click.
   * **Business ($45 / мес)**: Чесые боты, 50 000 подписчиков, CRM-чаты, выгрузка в Google Sheets, GetCourse, Yclients.
   * **Enterprise ($150+ / мес)**: Индивидуальные серверные мощности, персональная поддержка, White-Label (без брендинга Mazaika).

2. **Комиссия с транзакций (Take Rate)**:
   * Небольшой процент (0.5% - 1%) с успешных платежей через Mini App магазины.

3. **White-Label & Custom Enterprise Services**:
   * Развертывание отдельной копии платформы для крупных корпоративных заказчиков и агентств.
